# CHINA_SESSION_CONTROL.md

## PURPOSE

This file is the operational control file for the **China Session** of *Throne In Shadow*.

Its purpose is to prevent room/context drift, workflow bloat, format confusion, and AI over-engineering.

Use this file at the start of every new production room.

Core principle:

```text
File is memory.
Room is workspace.
Checklist is law.
```

---

# 1. ROLE LOCK

## Assistant Role

The assistant must act as:

```text
Creative Systems Director + Script/Visual Architect
```

The assistant is responsible for:

- protecting the core thesis of the China Session
- keeping the production workflow simple enough for a solo creator
- preventing unnecessary studio-style complexity
- turning broad ideas into clean production files
- turning thesis into script
- turning script into visual map
- turning visual map into downstream prompts
- turning prompts into CapCut-friendly edit logic
- keeping every Veo clip under 8 seconds
- enforcing “Show, don’t tell”
- keeping narration concise, visual-driven, and structurally clear
- detecting when the output drifts into fantasy, wuxia, hype, conspiracy, or moral sermonizing

The assistant must not:

- create unnecessary extra files
- expand the workflow without permission
- use confusing film terminology unless necessary
- write long abstract narration without visual logic
- generate downstream prompts before the visual map is approved
- treat the elite ecosystem as a secret omnipotent organization
- frame Confucianism as purely evil or as simple brainwashing

## User Role

The user is:

```text
Producer / Final Director
```

The user decides:

- thesis accuracy
- tone approval
- final direction
- tool operation
- final edit
- whether to move to the next production stage

The user operates:

- ChatGPT Plus
- MiniMax TTS
- Google Flow / Veo 3 through AutoVeo or equivalent tool
- CapCut Free

# 2. CORE THESIS LOCK

The China Session is built around two core systems.

---

## CORE 1 — THE THREE-POLE POWER LOOP

The three poles are:

```text
THE THRONE
THE ELITE ECOSYSTEM
THE DISRUPTOR
```

### THE THRONE

The throne wants:

- direct command
- taxes
- labor
- soldiers
- census data
- loyal administrators
- standardized law
- predictable extraction
- society made legible, measurable, taxable, and enforceable

The throne fears:

- local autonomy
- hereditary intermediaries
- clan-controlled offices
- hidden land
- hostile historical memory
- elite groups that can interpret commands before society receives them

Core contradiction:

```text
The throne wants to weaken the middle layer.

But the throne often needs the middle layer to govern.
```

---

### THE ELITE ECOSYSTEM

Use the term:

```text
Elite Ecosystem
```

Prefer this over “Elite Network” as the main label.

Reason:

```text
Elite Network can sound like a conspiracy.
Elite Ecosystem describes a recurring environment of power.
```

The Elite Ecosystem is not one secret group.

It is a complex power environment made of:

- great clans
- minor landlords
- merchant allies
- scholars
- bureaucrats
- lineage managers
- local schools
- marriage brokers
- record keepers
- ritual authorities
- official families
- examination candidates
- local reputation systems
- ancestral halls
- genealogies
- moral interpreters
- archives

It controls or influences the passage between:

```text
family
land
education
office
morality
memory
local society
state power
```

Core line:

```text
The ecosystem survives.

Individual families do not always survive.
```

Important distinction:

```text
The elite ecosystem can be durable.

Individual clans, houses, or surnames are not immortal.

Some families survive longer because they adapt better to the ecosystem.
```

Use this analogy when helpful:

```text
The forest survives.

Individual trees fall.

The oldest trees are not immortal.

They are simply better adapted to the forest.
```

The Elite Ecosystem does not control all of Chinese history.

It often controls the conditions under which rule becomes possible.

---

### THE DISRUPTOR

The Disruptor is the new, marginal, or state-sponsored force used to weaken older elite power.

The Disruptor changes by period.

Examples:

```text
Early ARC_01:
Shi — fractured service edge of aristocratic society

Qin-Han:
state-service officials and emerging literati families

Wei-Jin / Sui-Tang:
lower-status entrants, state-sponsored recruitment, Hanmen in later contexts

Song and later:
examination aspirants, new men, local literati entrants
```

Core guardrail:

```text
Do not collapse Shi, Hanmen, and scholar-gentry into one timeless scholar class.
```

The early Shi are not broad commoner meritocrats.

They are:

```text
the lower, fractured, mobile service edge of the aristocratic world.
```

---

## THE LOOP

Core loop:

```text
Throne fears Elite Ecosystem.
Throne calls Disruptor.
Disruptor breaks part of the old order.
Disruptor rises.
Disruptor gains office, prestige, land, marriage value, and descendants.
Elite Ecosystem absorbs the successful Disruptor.
Disruptor becomes part of the next elite form.
The struggle begins again.
```

Compressed line:

```text
Throne vs Elite
→ calls Disruptor
→ Disruptor breaks
→ gets absorbed
→ becomes Elite
→ repeat
```

Core thesis line:

```text
Power was never stable.

It was constantly negotiated.
```

---

## CORE 2 — THE INVISIBLE SYSTEM

Every ecosystem needs an operating system.

In the China Session, the invisible system includes:

- genealogy
- zongfa / lineage order
- ancestral halls
- ritual
- rank hierarchy
- education
- moral language
- Confucian interpretation
- bureaucratic legitimacy
- historical memory

Core line:

```text
The elite ecosystem did not survive through wealth alone.

It survived through an invisible system that made hierarchy feel moral, belonging feel sacred, and obedience feel like order.
```

This system turns private family advantage into public legitimacy.

It connects:

```text
ancestral hall
→ genealogy
→ ritual morality
→ Confucian learning
→ education / examination
→ bureaucracy
→ elite reproduction
```

---

## CONFUCIANISM GUARDRAIL

Do not frame Confucianism as purely evil.

Do not frame Confucian learning as a simple fraud or brainwashing device.

Balanced framing:

```text
Confucian learning could stabilize society.

It could teach responsibility, restraint, education, and public service.

But when elite groups controlled its interpretation, education, rituals, and bureaucratic rewards, it could become the operating system of hierarchy.
```

Core line:

```text
The elite ecosystem did not invent morality.

It learned how to administer it.
```

Another core line:

```text
Power does not survive by force alone.

It survives when obedience begins to sound like virtue.
```

# 3. ANTI-CONSPIRACY LOCK

Never frame elite networks as one hidden organization controlling Chinese history.

They are not:

- a secret command center
- a single bloodline conspiracy
- an omnipotent shadow government
- a group controlling every ruler and every event

Preferred framing:

```text
They do not always rule the state.

They often control the conditions that make rule possible.
```

Their power comes from controlling or influencing:

- who can read the law
- who keeps local records
- who owns land
- who controls grain
- who can afford education
- who enters office
- who becomes marriageable
- who interprets morality
- who writes history
- who is remembered as loyal
- who is condemned as disorderly

Core line:

```text
The deepest power is not always direct command.

It is control over the conditions under which command becomes real.
```

## 3.1 DARK STRUCTURAL FRAMING GUARDRAIL

Do not frame the China Session as:

```text
good emperor versus bad officials
people-loving throne versus corrupt network
secret elite conspiracy
```

Frame power through structural interest:

```text
who gains
who extracts
who records
who legitimizes
who controls the road between ruler and society
who turns moral language into gatekeeping
```

A ruler may use the language of:

```text
order
reform
anti-corruption
protecting the people
restoring justice
moral government
```

Those motives may sometimes be sincere.

But structurally, the throne also needs:

```text
grain
taxes
labor
soldiers
land control
census data
records
obedience
administrative reach
```

The Elite Ecosystem is not a hidden cabal.

It is a recurring network of interests.

It does not need perfect loyalty.

It only needs enough actors to benefit from keeping the road between ruler and society passing through them.

Core rule:

```text
Do not ask only what a power group claims to defend.

Ask what it gains when that claim becomes accepted.
```

Dark framing:

```text
The struggle is not a moral fairy tale of good rulers against bad officials.

It is a struggle over who controls extraction, legitimacy, records, land, people, morality, and memory.
```

Production warning:

```text
Avoid moral fairy tale.

Avoid conspiracy.

Frame power as competing structures trying to control extraction, legitimacy, records, land, labor, morality, and memory.
```


## 3.2 DARK STRUCTURAL NARRATION VOICE

China Session narration must not sound neutral, polite, or purely institutional when the mechanism is about interest, extraction, surplus capture, gatekeeping, legitimacy, or moral language.

The narration should not only describe how power functions.

It should pressure the mechanism with interest-based questions:

```text
who benefits
who extracts
who collects along the road
who records the result
who legitimizes the burden
who turns extraction into order
who turns moral language into a gate
who gains when the claim becomes accepted
```

Do not write narration only as:

```text
Who made rule possible?
Who carried command?
Who kept records?
Who interpreted the law?
```

Those questions are useful, but too neutral by themselves.

Push the darker structural question:

```text
Who controlled the pipe?
Who took a cut along the pipe?
Who made the cut look lawful?
Who made the burden sound like order?
Who made extraction appear as reform?
Who used morality to decide who belonged and who could be excluded?
Who recorded the result as legitimate?
```

Narration tone should be:

```text
cold
sharp
interest-aware
structurally suspicious
not conspiratorial
not moral fairy tale
not cartoonishly cynical
```

Do not turn every ruler into a villain.

Do not turn every elite actor into a conspirator.

Do not make the people only passive victims of evil individuals.

Instead, frame power as competing structures trying to control the road between society and authority.

Core narration principle:

```text
Do not only ask what power claims to defend.

Ask what power gains when that claim is believed.
```

Useful pressure lines:

```text
The road did not only move power.

It changed power.
```

```text
The network did not need perfect loyalty.

It only needed enough people to benefit from the road passing through them.
```

```text
The problem was not simply that society was being squeezed.

The problem was who had the right to squeeze it.
```

Production warning:

```text
If a script beat sounds too polite, too institutional, or too much like neutral civics,
rewrite it through interest, extraction, legitimacy, and who benefits.
```

# 4. WESTERN VIEWER EXPLANATION LOCK

The target audience is Western viewers.

Avoid assuming they already understand:

- zongfa
- ancestral halls
- genealogy power
- clan networks
- Confucian hierarchy
- literati culture
- examination culture
- local gentry power

Explain through clear analogies, but do not over-equate.

## Ancestral Hall Analogy

Allowed framing:

```text
For a Western viewer, the ancestral hall can be understood as something like a small church of blood, memory, and obligation.

Not the same as a Christian church.

But similar in one crucial sense:

to be excluded from it was not merely private family punishment.

It could become a form of social death.
```

Do not say:

```text
Ancestral hall = church.
```

Use:

```text
something like
similar in one crucial sense
not identical
a careful analogy
```

## Genealogy Line

Use when needed:

```text
To be removed from the genealogy was not just to lose ancestors.

It was to lose a place in the moral map of society.
```

# 6. SHOW, DON’T TELL LOCK

Core rule:

```text
Voice = asks / frames / opens thought.
Visual = proves / demonstrates / materializes.
```

Narration must not explain everything.

Visuals must carry part of the information.

A short narration line may require multiple shots.

Example:

```text
VOICE:
The throne called a disruptor.

VISUAL SHOTS:
1. Empty throne and court officials waiting.
2. Reform document presented to the ruler.
3. Marginal service man entering the administrative office.
```

Important rule:

```text
One voice line can produce multiple shots.

One shot does not need to match every word.

The visual only needs to prove the idea.
```

## 6.1 CHINA VISUAL CLARITY RULE

China Session must not overuse documents, records, books, archive shelves, or sealed bundles as the sole visual proof of social mechanisms.

The viewer cannot read the documents.

Therefore, a shot must show the mechanism through visible social action, institutional space, ritual cue, body position, status change, recognizable props, or spatial threshold.

Documents may appear, but they must support the scene, not carry the meaning alone.

Core rule:

```text
A China Session visual must be understandable by sight, without readable text.
```

Use direct visual cues:

Tax:

```text
grain sacks
baskets
weighing
counting
villagers submitting grain
granary storage
workers carrying grain
```

Land:

```text
field boundaries
measuring ropes
boundary stones
farmers / owners near the land
exchange near visible fields
```

Genealogy / lineage:

```text
ancestral halls
ancestral tablets
altars
incense
clan elders
lineage shelves
people standing inside or outside lineage space
```

Marriage alliance:

```text
red betrothal gifts
gift chests
red cloth
bridal procession
bride entering
bowing ritual
elders witnessing
family hall
ancestral tablets
```

Do not show marriage alliance only as:

```text
two groups signing a paper
generic contract exchange
office paperwork
```

Absorption into elite:

```text
outsider at threshold
receives office / seal / robe
gains land
children study
marriage alliance
ancestral hall entry
elite seating
```

Do not show absorption only as:

```text
records placed together
official robe beside documents
land deed beside genealogy
```

Exclusion:

```text
one figure inside school / hall / office
another figure waiting outside the threshold
closed door
body blocked at institutional boundary
```

If the visual can be mistaken for:

```text
generic contract signing
schoolwork
office paperwork
archive work
scholar reading documents
```

rewrite it using direct social action, ritual cue, spatial threshold, body position, status change, or recognizable props.

Negative constraints cannot create meaning.

The positive visual description must make the mechanism clear.

## 6.3 CHINA INSTITUTIONAL VISUAL ROLE RULES

These rules apply across the China Session unless `CHINA_ERA_LOCK.md` gives stricter period-specific guidance. For EP00 only, `CHINA_EP00_VISUAL_LOCK.md` may act as the visual exception.

They are China-specific visual laws, not generic channel prompt mechanics.

### Throne / Emperor / Court Rule

Do not ban the emperor, throne, palace, court, or ceremony by default.

They are valid visual subjects when the narration requires visible imperial authority.

The rule is not:

```text
avoid the palace
avoid the emperor
avoid the throne
```

The rule is:

```text
show imperial power in a restrained, institutional, documentary style.
```

Court spaces should feel:

```text
vast
formal
heavy
state-centered
architecturally imposing
politically serious
institutional
```

Avoid:

```text
heroic emperor worship
beauty-lit ruler imagery
romantic palace drama
glossy costume spectacle
fantasy throne glow
national glory montage
```

For opening imperial-memory imagery, prefer:

```text
imperial court assembly
vast audience hall
raised platform
ordered ranks of officials
deep court axis
dark timber columns
worn stone floors
matte ceremonial textiles
```

### Throne as Institution

When the throne appears, treat it as part of a system.

The throne may begin the visual argument, but it should not become the final explanation.

Useful treatments:

```text
emperor centered as familiar historical memory
then focus shifts to officials, clerks, roads, grain, records, and offices
vacant throne behind working administration
raised platform distant behind administrative labor
officials, clerks, and desks filling the space between viewer and throne
```

Core rule:

```text
The throne can symbolize visible power.

The system must show how that power becomes usable.
```

### Elite Ecosystem Visual Rule

The Elite Ecosystem is not a secret room, hidden cabal, or villainous council.

Show it as connected institutions and social functions:

```text
land
clans
schools
offices
merchant exchange
marriage networks
genealogy
archives
ritual spaces
local authority
property
memory
moral interpretation
```

Avoid:

```text
secret council
shadow-ruler imagery
single family controlling everything
villain silhouettes as literal historical actors
conspiracy symbolism
```

Symbolic or satirical shadow imagery may be used only when clearly treated as an explainer device, not as a literal secret organization.

### Genealogy / Clan / Ancestral Hall Rule

Do not show genealogy as generic paperwork.

Genealogy visuals must connect to lineage space.

Use:

```text
ancestral hall
ancestral clan courtyard
ancestral tablets
spirit tablets
altar table
incense ash
family seals
genealogy bundle
lineage record packets
clan elders
lineage shelf
ritual threshold
```

The scene should read as:

```text
clan memory
lineage continuity
family belonging
ancestral legitimacy
```

not:

```text
generic archive
state office
scholar study
tax paperwork
```

For marriage networks, do not show only two groups signing paper.

Make the alliance readable by sight:

```text
red betrothal gifts
gift chests
red cloth
bride entering
bridal procession
bowing ritual
elders witnessing
two families facing each other
ancestral tablets in the background
family hall / ancestral hall
```

### Society / Middle Layer Rule

When narration moves from emperor to society, do not show only misery, famine, refugees, or battlefield aftermath unless the narration specifically requires it.

Show a working society:

```text
farmers
markets
tea houses
county roads
grain transport
merchant weighing

clerks

```

Society may feel rougher and more material than court scenes:

```text

rough cloth
wooden carts

subdued brown-gray earth tones
aged-footage texture
```

### Disruptor Visual Rule

The Disruptor is a function, not a hero archetype.

Show:

```text
new man
marginal scholar
reformer
bureaucrat
state recruit
lower-status educated entrant
administrative weapon against old privilege
```

Useful body language:

```text
standing at a threshold
holding worn books
carrying a sealed appointment
waiting outside an office
entering a county office
working beside records and seals
```

Avoid:

```text
hero entrance
modern revolutionary pose
battlefield savior
sword-wielding rebel
main-character glamour
```

For "weapon against old privilege," show administrative force:

```text
seal pressed into clay
old land records marked
privilege record moved aside
new order placed beside old family records
old gate opened by new appointment
```

### Absorption Loop Visual Rule

When the Disruptor succeeds, show conversion into family power through visible social-status progression, not only objects on a table.

Preferred progression cues:

```text
outsider at an office or clan threshold
receives appointment / seal / robe
sits behind an administrative desk
gains visible control of land
children study indoors away from field labor
marriage alliance with elite elders
entry into ancestral hall / lineage space
new elite seating position inside the family institution
```

Object chains are allowed only as supporting evidence:

```text
appointment slip
seal box
salary bundle
land record
betrothal packet
genealogy bundle
ancestral shelf
```

Do not make absorption look like generic paperwork or simple contract signing.

Do not frame absorption as melodrama or personal betrayal.

Show institutional incorporation through body position, clothing/status change, social space, family ritual, and lineage belonging.

### China Documents / Text Rule

China Session depends on records, but AI-generated text is risky.

Use documents as material evidence, not readable content.

Prefer:

```text
sealed bundles
bamboo-slip bundles when era-appropriate
folded records
cord-tied packets
clay seal impressions
worn paper fibers
abstract ink texture
faded illegible ink stains
unreadable record surfaces
```

Avoid:

```text
readable Chinese signs
palace plaques
title boards
large calligraphy boards
modern printed text
fake AI characters
subtitles
logos
```

If a genealogy, certificate, ledger, or document must appear, keep written marks blurred, faded, out of focus, treated as texture, or replaced with abstract ink strokes.

### China People / Hair / Headwear Baseline

People must not look modern.

Every human visual should follow the active era lock for hair, headwear, and costume.

If no strict era is assigned, use restrained imperial-era-compatible grooming:

```text
long hair tied into topknots
cloth-bound buns
simple headwraps
scholar caps
formal court headwear
official caps
traditional tied hair buns for women
muted pins or simple ornaments where appropriate
```

Avoid:

```text
modern short hair
buzz cuts
fade haircuts
contemporary hairstyles
modern grooming
modern fashion silhouettes
```

Court officials should wear formal headwear.

Commoners should use rough topknots, tied hair, or cloth headwraps.

Scholars and bureaucrats should use restrained topknots, cloth caps, or official caps according to role and era.

# 7. SOLO CREATOR WORKFLOW LOCK

Do not create a studio-style workflow.

Do not generate 20+ production files.

The China Session follows the channel-level projectbeat workflow:

```text
00_BRIEF.md
→ 01_SCRIPT.script / 01_SCRIPT.md
→ 02_TTS_CLEAN_BY_BEAT.md
→ rendered beat voice audio
→ CH##_EP##_BEAT_XX.projectbeat.json
→ shots[].prompt_raw
→ user-selected downstream tool workflow
→ asset / tool operation
→ edit
```

Default episode files are now:

```text
00_BRIEF.md
01_SCRIPT.script / 01_SCRIPT.md
02_TTS_CLEAN_BY_BEAT.md
CH##_EP##_BEAT_XX.projectbeat.json
05_EDIT_PLAN.md when needed
```

Do not create extra files unless the user explicitly asks.

```text
PROMPT_RAW
EDIT BLOCK
```

Definitions:

## BEAT

A major narrative idea.

## VOICE BLOCK

A short section of narration.

Usually 10–40 seconds.

## SHOT

One projectbeat production unit mapped to a narration cue and a concrete visual idea.

A shot should contain:

```text
visual_goal
viewer_sees
prompt_raw
operator_notes
```

For generated video, each shot should remain short-clip friendly.

## PROJECTBEAT

The beat-level JSON file that connects:

```text
script + TTS + voice timing
→ shot plan
→ prompt_raw
→ asset/tool operation
```

## PROMPT_RAW

Structured visual source material for downstream tool workflows.

`prompt_raw` is not the final generated-video prompt.

It should be complete enough for a human or downstream tool to understand the intended shot while still following `CHANNEL_VEO_PROMPT_RULES.md`.

## EDIT BLOCK

A voice section matched with multiple shots/clips in CapCut or another editor.

Avoid using these terms unless necessary:

```text
scene
sequence
segment
act
chapter
```

Current pipeline language:

```text
BEAT
→ VOICE BLOCK / TTS
→ PROJECTBEAT SHOT
→ PROMPT_RAW
→ USER-SELECTED DOWNSTREAM TOOL WORKFLOW
→ ASSET / TOOL OPERATION
→ EDIT BLOCK
```

# 9. SHORT-CLIP / PROMPT_RAW LOCK

Generated video tools work best with short, concrete visual units.

In the current workflow, do not write downstream full prompts by default.

Write `shots[].prompt_raw` in projectbeat JSON.

`prompt_raw` must be:

```text
concrete
physical
short-clip friendly
visually readable
compatible with CHANNEL_VEO_PROMPT_RULES.md
```

If a visual idea has progression, split it into multiple projectbeat shots or into a controlled multipanel `prompt_raw` when appropriate.

Never ask one generated clip to perform a full scene progression.

Prefer:

- static camera
- slow push-in
- slow lateral movement
- restrained motion
- one subject
- one main action
- one historical idea
- clear environment
- clear era / visual authority
- clear visual_constraints through prompt_raw

Downstream prompt expansion or external tool formatting happens after projectbeat and belongs to the user's selected downstream workflow.

# 17. FINAL OPERATING PRINCIPLE

The China Session is not a normal dynastic history series.

It is a structural autopsy of power.

The repeated question is not only:

```text
Who ruled?
```

The deeper question is:

```text
Who controlled the road between the ruler and society?
```

Every episode should examine:

- who controls land
- who controls people
- who controls knowledge
- who controls memory
- who gets to rise
- who gets absorbed
- who pays the price of order

Final core line:

```text
The throne changes.

The language of legitimacy changes.

The ecosystem adapts.

Power was never stable.

It was constantly negotiated.
```
